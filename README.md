# PVT-15 Backend

Spring Boot backend for the PVT-15 project.

## Requirements
- IntelliJ IDEA
- Java 21
- Git

## Setup
1. Clone the repository
2. Open the project in IntelliJ
3. Reload the Maven project
4. In the run configuration, add these environment variables:
   - `DB_URL=jdbc:mariadb://mysql.dsv.su.se:3306/your_database`
   - `DB_USER=your_username`
   - `DB_PASS=your_password`
5. Run `AccessingDataMysqlApplication`

## Local URL
- `http://localhost:8080`
